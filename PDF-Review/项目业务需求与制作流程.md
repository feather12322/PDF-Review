# AI 赋能的智能简历分析系统 - 业务需求与制作流程

## 一、项目概述

### 1.1 项目背景
在招聘流程中，HR 需要处理大量简历，人工筛选效率低下且容易遗漏优质候选人。本系统通过 AI 技术自动解析简历、提取关键信息并进行智能匹配，大幅提升招聘效率。

### 1.2 项目目标
- 自动化简历解析，减少人工录入工作
- 智能提取关键信息，结构化存储候选人数据
- 基于岗位需求进行智能匹配和评分
- 提供友好的 Web 界面，支持在线操作

### 1.3 技术架构
- **后端**：Python + 阿里云 Serverless（函数计算 FC）
- **前端**：现代前端框架（React/Vue）+ GitHub Pages 部署
- **API 规范**：RESTful
- **缓存**：Redis（可选加分项）
- **AI 能力**：大语言模型（如通义千问、GPT 等）

## 二、业务需求详解

### 2.1 模块一：简历上传与解析（必选）

#### 业务需求
- 用户可通过 Web 界面上传单个 PDF 格式简历
- 系统自动解析 PDF 文件，提取全部文本内容
- 支持多页简历的完整解析
- 对提取的文本进行清洗和预处理

#### 功能要点
1. **文件上传**
   - 限制文件格式：仅支持 PDF
   - 文件大小限制：建议 10MB 以内
   - 上传方式：HTTP POST multipart/form-data

2. **PDF 解析**
   - 使用 Python 库（如 PyPDF2、pdfplumber）提取文本
   - 处理多页简历，合并所有页面内容
   - 保留文本的基本结构（段落、换行）

3. **文本清洗**
   - 去除多余空格、特殊字符
   - 统一编码格式（UTF-8）
   - 合理分段，便于后续信息提取

#### 接口设计
```
POST /api/resume/upload
Content-Type: multipart/form-data

Request:
- file: PDF 文件

Response:
{
  "code": 200,
  "message": "上传成功",
  "data": {
    "resume_id": "uuid",
    "raw_text": "提取的原始文本",
    "page_count": 2
  }
}
```

### 2.2 模块二：关键信息提取（必选）

#### 业务需求
利用 AI 模型从简历文本中智能提取结构化信息，包括基本信息、求职信息和背景信息。

#### 信息提取字段

| 类别 | 字段 | 优先级 | 说明 |
|------|------|--------|------|
| 基本信息 | 姓名 | 必选 | 候选人姓名 |
| 基本信息 | 电话 | 必选 | 联系电话 |
| 基本信息 | 邮箱 | 必选 | 电子邮箱 |
| 基本信息 | 地址 | 必选 | 居住地址 |
| 求职信息 | 求职意向 | 加分项 | 期望岗位 |
| 求职信息 | 期望薪资 | 加分项 | 薪资范围 |
| 背景信息 | 工作年限 | 加分项 | 工作经验年数 |
| 背景信息 | 学历背景 | 加分项 | 最高学历、毕业院校、专业 |
| 背景信息 | 项目经历 | 加分项 | 项目名称、职责、技术栈 |

1. **AI 模型调用**
   - 使用大语言模型（LLM）进行信息提取

   - 返回结构化 JSON 数据

### 2.3 模块三：简历评分与匹配（必选）

#### 业务需求
根据岗位需求描述，对简历进行智能匹配和评分，帮助 HR 快速筛选合适候选人。

#### 功能要点
1. **岗位需求分析**
   - 接收岗位描述文本（职位名称、技能要求、工作经验等）
   - 提取岗位关键词（技能、工具、行业经验）
   - 分析岗位优先级和权重

2. **匹配度计算**
   - **技能匹配**：简历技能与岗位要求的重合度
   - **经验匹配**：工作年限、行业经验的相关性
   - **学历匹配**：学历要求的符合程度
   - **综合评分**：加权计算总分（0-100 分）

3. **AI 增强评分（加分项）**
   - 使用 LLM 进行语义理解
   - 分析简历与岗位的深层匹配度
   - 生成匹配理由和建议

#### 评分算法示例
```python
# 基础评分算法
skill_score = len(resume_skills & job_skills) / len(job_skills) * 40
experience_score = min(resume_years / required_years, 1.0) * 30
education_score = education_level_match * 20
other_score = 10

total_score = skill_score + experience_score + education_score + other_score
```

#### 接口设计
```
POST /api/resume/match
Content-Type: application/json

Request:
{
  "resume_id": "uuid",
  "job_description": "招聘 Python 后端工程师，要求 3 年以上经验..."
}

Response:
{
  "code": 200,
  "message": "匹配成功",
  "data": {
    "resume_id": "uuid",
    "match_score": 85,
    "details": {
      "skill_match": 90,
      "experience_match": 80,
      "education_match": 85
    },
    "matched_keywords": ["Python", "Django", "MySQL", "Redis"],
    "suggestions": "候选人技能匹配度高，建议安排面试"
  }
}
```

### 2.4 模块四：结果返回与缓存（必选）

#### 业务需求
- 所有接口返回统一的 JSON 格式
- 实现缓存机制，避免重复计算（加分项）

#### 功能要点
1. **统一响应格式**
```json
{
  "code": 200,
  "message": "操作成功",
  "data": {},
  "timestamp": 1234567890
}
```

2. **缓存策略（加分项）**
   - 使用 Redis 缓存已解析的简历
   - 缓存键设计：`resume:{resume_id}:parsed`
   - 缓存过期时间：24 小时
   - 缓存匹配结果：`resume:{resume_id}:match:{job_hash}`

3. **缓存流程**
```
1. 收到请求 → 检查缓存
2. 缓存命中 → 直接返回
3. 缓存未命中 → 执行解析 → 存入缓存 → 返回结果
```

#### 步骤 2：实现 PDF 解析服务

```python
# app/services/pdf_parser.py
import pdfplumber

class PDFParser:
    def parse(self, file_path):
        """解析 PDF 文件"""
        text = ""
        with pdfplumber.open(file_path) as pdf:
            for page in pdf.pages:
                text += page.extract_text() + "\n"
        
        # 文本清洗
        text = self.clean_text(text)
        return text
    
    def clean_text(self, text):
        """清洗文本"""
        # 去除多余空格
        text = " ".join(text.split())
        # 其他清洗逻辑
        return text
```

#### 步骤 3：实现 AI 信息提取

```python
# app/services/ai_extractor.py
import openai

class AIExtractor:
    def __init__(self, api_key):
        self.client = openai.OpenAI(api_key=api_key)
    
    def extract_info(self, resume_text):
        """提取简历关键信息"""
        prompt = f"""
        请从以下简历中提取关键信息，以 JSON 格式返回：
        
        {resume_text}
        
        返回格式：
        {{
          "basic_info": {{"name": "", "phone": "", "email": "", "address": ""}},
          "job_intention": {{"position": "", "salary": ""}},
          "background": {{"work_years": 0, "education": "", "projects": []}}
        }}
        """
        
        response = self.client.chat.completions.create(
            model="gpt-3.5-turbo",
            messages=[{"role": "user", "content": prompt}],
            response_format={"type": "json_object"}
        )
        
        return response.choices[0].message.content
```

#### 步骤 4：实现匹配评分

```python
# app/services/matcher.py
class ResumeMatcher:
    def match(self, resume_data, job_description):
        """计算匹配度"""
        # 提取岗位关键词
        job_keywords = self.extract_keywords(job_description)
        
        # 计算技能匹配度
        skill_score = self.calculate_skill_match(resume_data, job_keywords)
        
        # 计算经验匹配度
        exp_score = self.calculate_experience_match(resume_data, job_description)
        
        # 综合评分
        total_score = skill_score * 0.5 + exp_score * 0.3 + 20
        
        return {
            "match_score": round(total_score, 2),
            "details": {
                "skill_match": skill_score,
                "experience_match": exp_score
            }
        }
```

### 4.1 PDF 解析
- 推荐库：`pdfplumber`（准确度高）或 `PyPDF2`（轻量级）
