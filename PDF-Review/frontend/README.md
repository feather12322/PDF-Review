# 智能简历分析系统 - 前端

基于 Vue 3 + Element Plus 的现代化 Web 应用。

## 功能特性

- 简历拖拽上传
- 实时信息展示
- 可视化匹配评分
- 响应式设计

## 快速开始

### 1. 安装依赖

```bash
cd frontend
npm install
```

### 2. 开发模式

```bash
npm run serve
```

应用将在 http://localhost:8080 启动

### 3. 生产构建

```bash
npm run build
```

构建产物在 `dist` 目录

## 部署到 GitHub Pages

1. 修改 `vue.config.js` 中的 `publicPath`
2. 运行 `npm run build`
3. 将 `dist` 目录内容推送到 GitHub Pages

## 技术栈

- Vue 3
- Vue Router
- Axios
- Element Plus
- ES6+
